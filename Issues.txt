the error is due to a compatibility issue between Flask-MongoEngine and newer versions of Flask. Flask removed JSONEncoder in version 2.2+, but Flask-MongoEngine hasn't been updated yet.
then install this 

pip install "Flask<2.2" "Werkzeug<2.2"
OR
pip install Flask==2.1.3 Werkzeug==2.1.2

npm run start:tunnel
